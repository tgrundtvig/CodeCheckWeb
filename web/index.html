Minimal Code Checker Prototype

This is a small 'proof of concept' code checking web server. The idea is that it should be able to auto-correct programming-assignments written by students.

WARNING: Work in progress.

This is only the very first prototype. There is no real security or even an administrative module yet. Users and assignments are inserted directly in the database using sql.
